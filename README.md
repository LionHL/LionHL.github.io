### Usage
```git 
git clone git@github.com:LionHL/LionHL.github.io.git

git checkout hexo
```

hexo commands:
```git

hexo d  //hexo deploy 将.deploy目录部署到GitHub
```
   
```git 
git add .
git commit -m '...'
git push origin hexo
```
